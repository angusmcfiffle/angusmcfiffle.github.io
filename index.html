Suika Game Clone

A version of Suika Game (or watermelon game) that has different score scaling to the original. In this game, you drop different fruits on each other. If 2 or more of the same fruit touch, they will combine into a larger fruit. The goal is to get as many points as possible, while not overfilling the screen with fruit.
